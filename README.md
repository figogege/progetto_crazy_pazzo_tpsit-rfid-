# progetto_crazy_pazzo_tpsit-rfid-
progetto tpsit D'amore Gunebakmaz zadro. Progetto per lo sviluppo di un sistema che sfrutta la tecnologia rfid e un database.

Il propgetto consiste nello sviluppo di un sistema per un eventuale azienda che fornisce servizi in giro per le citta (es biclette,powerbank o altro) per poter usufruire dei servizi è necessario pagare un abbonamento, tutti gli abbonati sono inseriti in un server.
Il progetto sfrutta anche un sistema rfid per poter accedere ai servizi. L'abbonato dovrà avvicinare una carta con l'rfid fornita dall'azienda, la carta contiene tutti i dati dell'utente, si andrà cosi a verificare  se l'utente è nel sistema del database e potrà quindi accedere.
Una possibile espansione del sistema è quella di inserire diversi tipi di abbonamenti che definisco per quante ore un utente puo accedere al servizio. Un altro possibile servizio aggiuntivo è quello di creare una sorta di tabelle con le varie statistiche degli abbonati per poter fornire alle aziende le varie statisctiche riguardo a chi utilizza maggiormente i servizi (es maschio/femmina, l'età, il luogo in cui utilizza il servizio, per quanto lo usa e cosa usa maggiormente)
